# **App Name**: SkinAI Advisor

## Core Features:

- Skin Analysis: Allow users to upload a photo of their face for AI analysis, and analyze a user's uploaded image to determine the condition of the users skin, identifying specific skin concerns.
- Report Generation: After processing an uploaded image, the LLM tool will generate a detailed report assessing skin condition.
- Product Carousel: Display a carousel with a listing of recommended skincare products tailored to the user's identified skin concerns.
- Save Product Recommendations: Allows the user to save or favorite preferred product recommendations locally in the browser, with no external database required

## Style Guidelines:

- Primary color: Soft light green (#90EE90) for a fresh, natural aesthetic.
- Background color: Off-white (#FAFAFA) to create a clean and open feel.
- Accent color: Pale Yellow (#F0E68C) for highlights and interactive elements.
- Body and headline font: 'Inter' sans-serif, for a neutral, modern, and clean aesthetic
- Use minimalist line icons to represent different skin concerns and product categories.
- Implement a clean, card-based layout for product recommendations and assessment results.
- Subtle transitions and animations when navigating between analysis results and product suggestions.
